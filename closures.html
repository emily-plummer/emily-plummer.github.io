I've set off on my Javascript Journey this week! I'm using Codeschool to learn the basics, they've actually got a pretty robust amount of courses to complete the Javascript Class. Right now I'm on Part Three of the Javascript series, after this I'll move on to learning JQuery, Node, Backbone & Angular which are all included courses on the Javascript path. Today the majority of my focus was spent on closures. They can be a little tricky to wrap your head around at first, but I think I'm starting to get the hang of it. Codeschool's approach is to get students good at being able to trace closures, I found task 2.4 to be quite challenging. Explaining the steps I went through helps me to cement what I've learned, so I'm going to explain my solution here.

Here is the function given, the module asks you to alert to the console the result of the function

var hidden = mystery(3);
var jumble = mystery3(hidden);
var result = jumble(2);

	function mystery(input) {
		var secret = 4;
		input +=2;
		function mystery2(multiplier) {
			multiplier *= input;
			return secret * multiplier;
	}
	return mystery2;
}
function mystery3(param) {
	function mystery4(bonus) {
		return param(6) + bonus;
}
return mystery4;
}

Step 1: The hidden variable is declared and assigned mystery(3); This means that mystery is assigned a parameter of 3. Input now equals 3, but by the time we get to the inner function, input equals 5.

Mystery2 now equals: 4 * (multiplier*5);

Step 2: The variable jumble is declared and assigned mystery3(hidden). The variable param in mystery(3) is now assigned ‘hidden’ which evaluates to the return of mystery, which is mystery2.

Step 3: The variable result is declared and assigned jumble(2). This sets the value of bonus to be 2. The function mystery3 now looks as follows:

function mystery3 (mystery2){
	function mystery4(2) {
		return param(6) + 2;
}

Step 4: Evaluating the return of mystery4 : param(6) sends the value of 6 to mystery2. This means multiplier is now assigned a value of 6, so mystery2 can be evaluated.

mystery2(6) {
	6 * 5;
	return 4* 30
}

(Returns 120).

This now passes to parameter for mystery3 and now mystery3 can be evaluated.

mystery3(120){
	function mystery4(2){
		return 120 + 2;
}

This evaluates to 122!
